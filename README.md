# SportNoteBook

A progressive web app (PWA) for tracking sports sessions, goals, and personal performance — designed with an iOS-style interface, installable on any device, and fully offline-capable.

## Features

- Session logging with sport type, duration, intensity, and notes
- Personal goal tracking
- Offline support via service worker
- Installable as a native app on iOS and Android
- All data stored locally on device (no server, no account)
- Clean, minimal interface — no ads, no tracking

## Live App

[calvinknp.github.io/SportNoteBook](https://calvinknp.github.io/SportNoteBook)

## Tech stack

Single HTML file — HTML, CSS, JavaScript — no frameworks, no build step.
